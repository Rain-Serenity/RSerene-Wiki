Basic Settings

Server Core

Warning

Please avoid excessive machine density. Thank you for your cooperation!

This server uses the Leaveopen in new window core.
Most vanilla features are enabled.

No Item Drop on Death

We have enabled no item drop on death.

Sleep System

We do not use a sleep acceleration mechanism but instead employ a custom sleep system using the NoPhantomSleepopen in new window plugin. The specifics are as follows: as long as you right-click a bed at night and sleep for more than 5 seconds, it will count as having passed the entire night. No phantoms will spawn within a 64-block radius around you, and this effect lasts for 30 minutes.

View Distance

Our view distance is set to 16 chunks.

Land Claim

The tool for creating land claims is set to a wooden hoe. The maximum claim size is 512×512 (X×Z), with no height limit (-64~328 is allowed).
