サーバールール

I. 総則

  1. 非営利性

    • Minecraft Rain Serenity Server は非営利の無料Minecraft Java Editionサーバーです。

  2. 寄付ポリシー

    • このサーバーには有料アイテムやPay-to-Win要素はありませんが、無条件の自主的な寄付を受け付けています。

  3. RMT免責事項

    • このサーバーはプレイヤーがゲーム内アイテムを現実世界の通貨と交換することを制限しませんが、サーバー運営チームはプレイヤー間の現実世界の経済的紛争に対して一切の責任を負いません。

  4. 紛争解決

    • サーバーでのプレイ中に問題、紛争、または議論が発生した場合は、冷静かつ合理的に対処し、必要に応じて管理者に連絡して対応を依頼してください。

  5. コミュニティの尊重

    • 他のプレイヤーを尊重し、すべての人に親切に接してください。

  6. 年齢ガイドライン

    • Mojangの公式ガイドラインによると、Minecraftの推奨最低年齢は9歳以上です。これ以下の年齢の場合は、保護者の監督下でプレイしてください。

II. 禁止行為

ペナルティ通知

違反が発見された場合、重大度に応じて 警告、一時的なBAN、または永久的な削除 の対象となります。

1. チャット＆コミュニティ

  • 不適切な発言
    • 差別的、攻撃的、または著しく不適切な発言をすること。 (下記セクションIIIを参照)
  • 不快なコンテンツ
    • ポルノ、不快、または過度に暴力的なコンテンツを投稿すること。
  • スパム
    • チャットにスパムや過剰なメッセージを送信すること。
  • 不適切な行為
    • 他人になりすますこと、許可なく広告を行うこと、またはその他不適切な行為を行うこと。

2. ゲーム内行動

チート

  • a. チートクライアント、ハック、または禁止されたModを使用すること。
  • b. サーバーやバニラゲームの脆弱性を悪意を持って悪用すること。

グリーフィング

  • a. 他のプレイヤーの建造物を無断で破壊または改変すること。
  • b. 他のプレイヤーのコンテナからアイテムを盗むこと。
  • c. 相互同意なしの悪意のあるPvPまたはプレイヤーキル（PK）。

メカニクスの悪用

  • a. プラグイン機能の悪用（例：他人の建造物を保護で囲う）。
  • b. サーバーコマンドのスパム送信。

違法建造物

  • 不適切、わいせつ、または不快な建造物を建てること。

III. ゼロトレランス違反

絶対的なレッドライン

以下の行為は 事前警告なしの即時永久BANおよびコミュニティグループからの削除 となります。これらの罰則は最終的なものであり、異議申し立ては一切受け付けません。

ヘイトスピーチ＆ハラスメント

違反説明
人種差別人種的中傷や人種差別的ヘイトスピーチ。
性差別性差別的発言やジェンダーに基づく差別。
反LGBTQ+同性愛嫌悪、トランスフォビア的な中傷、または反LGBTQ+スピーチ。
障害者差別障害者を標的にした中傷やヘイトスピーチ。
その他のヘイトスピーチ上記に記載されていないその他の形態のヘイトスピーチ。
個人情報暴露他人の現実世界の個人情報を暴露、漏洩、または配布すること。
深刻な嫌がらせ深刻な個人攻撃または有害なハラスメント。

IV. BAN異議申し立て

標準的な異議申し立てプロセス

  • 「ゼロトレランス違反」以外の違反によるBANに同意できない場合は、公式チャネルを通じて管理者に異議申し立てを提出し、罰則の軽減またはBAN解除をリクエストできます。
