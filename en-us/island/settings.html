Basic Settings

Server Core Configuration

Warning

Please avoid excessive machine density. Thank you for your cooperation!

This server uses the Paperopen in new window core.
Most vanilla duplication mechanics are enabled (a few are currently unavailable, pending Leaves update).

Island Settings

Each player's island is 512×512Island biomes can be changed (see command list for details)

Additional Crafting Recipes

/rim recipemenu to view
Due to the large number of recipes, they are not listed here.

No Item Drop on Death

We have enabled keep inventory on death.

Sleep System

Implemented via the NoPhantomSleepopen in new window plugin. As long as you right-click a bed at night, it will count as having passed the entire night, and no phantoms will spawn around you.

View Distance

Our view distance is set to 12 chunks.

Obtaining Sunflowers

In sunflower plains biomes, using bone meal on grass blocks has a 10% chance to replace 30% of flowers and grass within a 9×9 area with sunflowers.

Obtaining Coral

When the biome is warm ocean, using bone meal on underwater blocks can randomly generate coral (blocks, fans)
Probabilities:

ItemProbability
Sea Grass60%
Coral Block15%
Coral Plant15%
Coral Fan10%

Natural Guardian Spawning

In deep cold ocean biomes, natural squid/cod/salmon/pufferfish/tropical fish spawns are automatically replaced with guardians.

JEI Support

This server uses the custom plugin JEIBukkitopen in new window, which automatically syncs recipes to your client when you join.

Slime Chunks

/rim slime to check nearby slime chunks
You can also look them up manually
Server seed: -3483072257881485378

Sit Anywhere

Sneak at the edge of a block until you hear feedback, then release to sit down.

Iron Block Elevator & Rail Teleport

Iron Block Elevator: Place iron blocks at different Y levels with the same XZ coordinates, with enough space above. Sneak on the iron block to teleport down, jump to teleport up.Rail Teleport: Sneak on rails to teleport to the other end of the rail.

Cooperative Progression

The server supports cooperative development between friends.
/plot trust to invite friends to your island
/plot visit to visit other players' islands
