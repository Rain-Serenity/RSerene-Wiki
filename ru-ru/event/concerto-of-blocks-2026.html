Концерт блоков

Поддерживаемая версия: Minecraft Java Edition 26.2 (Chaos Cubed)
Сервер: RSerene ｜ IP: Play.RSerene.com ｜ Ванильное выживание или Остров
День оценки: 16 августа 2026 г.
Это мероприятие не является официальным мероприятием Minecraft и не связано с Mojang / Microsoft

🎯 Тема и концепция

«Концерт блоков» — Ванильный музыкальный фестиваль — Участники должны использовать только ванильные механики игры для создания музыкальных произведений. Задача: насколько впечатляющим может быть выступление с использованием нотных блоков + редстоуна + встроенных звуков игры?

Почему только ванильно? RSerene — это чисто некоммерческий, ванильный сервер выживания, который не изменяет геймплей. Этот фестиваль призван показать, «на что способна ванильная версия».

Три столпа ванильной музыки:

  1. Нотный блок (Note Block): Крафтится из досок + красной пыли. ПКМ для повышения тона на полутон. 25 нот (F#3 → F#5, две октавы). Над блоком должен быть воздух для издавания звука. Звук распространяется до 48 блоков.
    Тембр зависит от блока под ним: Древесина → Бас, Песок/Гравий → Малый барабан, Стекло → Щелчок, Камень → Большой барабан, Золотой блок → Колокольчики, Глина → Флейта, Плотный лёд → Колокола, Шерсть → Гитара, Костяной блок → Ксилофон, Железный блок → Вибрафон, Песок душ → Колокол, Тыква → Диджериду, Изумрудный блок → Квадратная волна, Сноп сена → Банджо, Светокамень → Электрическое пианино, Земля/Воздух → Пианино/Арфа
  2. Редстоун-исполнение: Используйте кнопки, нажимные пластины, рычаги, редстоуновые факелы, повторители (задержка = ритм) для управления цепями, заставляя несколько нотных блоков играть последовательно, создавая мелодии и аккорды
  3. Ванильные звуки: Новый серный куб в 26.2 поглощает разные блоки и меняет физические свойства, что может служить сценической механикой или интерактивными установками — или вы можете выбрать другие ванильные звуки блоков.

📋 Правила участия

Квалификация

  • Все игроки сервера RSerene могут участвовать
  • Допускается индивидуальное или командное участие (команды ≤ 4 игроков)
  • Свяжитесь с @Chosen_1st в QQ группе или отправьте личное сообщение Chosen_1st для регистрации

Предупреждение

Если после регистрации вы не участвуете или не выполняете технические требования, с вашего игрового баланса будет списано 999,99.

Технические требования

  1. 100% Ванильно: Разрешены только нотные блоки, компоненты редстоуна и встроенные звуки игры (например, проигрыватель с ванильными пластинками, звуки мобов). Запрещены моды, плагины, внешние аудиофайлы и ресурс-паки, заменяющие звуки
  2. Длительность выступления: 30 секунд — 5 минут
  3. Работа должна полностью воспроизводиться в ванильной среде 26.2
  4. Поощряются оригинальные композиции; при каверах на существующие песни участники должны самостоятельно обеспечить права на использование музыки
  5. Сценическое строительство разрешено с использованием ванильных блоков для визуального оформления

Запрещено

  • Моды или ресурс-паки, изменяющие звуки
  • Копирование чужих редстоун-схем без указания авторства
  • Злонамеренное вмешательство в работу других участников
  • Нарушение любого из вышеперечисленных пунктов влечёт немедленную дисквалификацию и меры согласно Правилам сервера
