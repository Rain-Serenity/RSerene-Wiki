## 🎯 이벤트 테마 및 콘셉트

**「블록 협주곡」바닐라 음악 축제** — 참가자는 **게임의 바닐라 메커니즘만 사용**하여 게임 내에서 음악 작품을 선보입니다. Note Block + Redstone + 게임 내장 사운드로 얼마나 놀라운 퍼포먼스를 만들 수 있는지 겨루는 대회입니다.

왜 바닐라만 고집하나요? RSerene은 **순수 공익, 순수 바닐라 서바이벌** 서버로, 게임플레이를 변조하지 않습니다. 이 음악 축제는 "바닐라로 얼마나 할 수 있는가"를 극한까지 끌어올리는 데 목적이 있습니다.

**바닐라 음악의 세 가지 요소**:

1. **Note Block**: 나무 판자+레드스톤 가루로 제작. 우클릭으로 반음 올림. **25음높이(F#3 → F#5, 2옥타브)**. 위에 공기가 있어야 소리가 나며, 최대 48블록까지 전달됩니다.  
   **음색은 아래 블록에 따라 결정**: 나무→베이스, 모래/자갈→스네어 드럼, 유리→클릭, 돌→베이스 드럼, 금 블록→벨, 점토→플루트, 꽁꽁 언 얼음→차임, 양털→기타, 뼈 블록→실로폰, 철 블록→비브라폰, 영혼 모래→카우벨, 호박→디저리두, 에메랄드 블록→구형파, 건초 더미→밴조, 발광석→일렉트릭 피아노, 흙/공기→피아노/하프
2. **레드스톤 연주**: 버튼, 압력판, 레버, 레드스톤 횃불, 리피터(지연=리듬)로 회로를 제어하여 여러 Note Block이 순서대로 소리를 내 멜로디와 코드를 만듭니다
3. **바닐라 사운드 효과**: 26.2에 추가된 **황산염 큐브**는 블록을 흡수하여 물리적 특성이 변합니다. 직접 소리를 낼 수는 없지만, 무대 기계/인터랙티브 장치의 일부로 활용할 수 있습니다.
