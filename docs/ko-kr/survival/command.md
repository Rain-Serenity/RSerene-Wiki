# 명령어 목록
| 명령어                             | 기능                            | 비고              |
|----------------------------------|-------------------------------|-----------------|
| /back                            | 마지막 저장 위치로 돌아가기              |                 |
| /balance                         | 잔액 확인                        |                 |
| /baltop                          | 잔액 순위 보기                     |                 |
| /co i                            | 블록 기록 조회 모드 켜기(끄기)                 |
| /dback                           | 마지막 사망 위치로 돌아가기              |                 |
| /gmp mailbox                     | 우편함 열기; 글로벌 마켓에서 내린 아이템이 보관됨 |                 |
| /gmp market gui                  | 글로벌 마켓 열기                    |                 |
| /gmp market sell <가격> <수량>     | 손에 든 아이템을 지정 가격으로 글로벌 마켓에 등록 |                 |
| /money                            | 잔액 확인                        |                 |
| /msg                             | 플레이어에게 귓속말 보내기               |                 |
| /pay <플레이어명> <금액>          | 플레이어에게 송금                     | 수수료 3%         |
| /ping                            | 지연 시간 확인                      |                 |
| /playtime                        | 플레이 시간 확인                     |                 |
| /playtimetop                     | 서버 플레이 시간 순위 보기               |                 |
| /removehome <집 이름>              | 집 삭제                          |                 |
| /res help                        | 영토 플러그인 도움말 보기                |                 |
| /rtp                             | 무작위 텔레포트                      | 범위 12000*12000 |
| /sell gui                        | 판매 메뉴 열기                     | 판매 가능 아이템은 "기본 설정" 참조 |
| /spawn                           | 스폰으로 텔레포트                     |                 |
| /tpa                             | 다른 플레이어에게 텔레포트 요청             |                 |
| /tpahere                         | 다른 플레이어가 자신에게 텔레포트하도록 요청      |                 |
| /worth                           | 손에 든 아이템의 판매 가치 확인             |                 |
| /worthlist                       | 판매 가능한 모든 아이템의 가치 보기           |                 |
