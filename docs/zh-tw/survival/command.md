# 指令列表
| 指令名稱                       | 功能                        | 備註            |
|----------------------------|---------------------------|---------------|
| /back                      | 返回上一個保存的地點                |
| /balance                   | 查看你的餘額                    |               |
| /baltop                    | 查看餘額排行榜                   |               |
| /dback                     | 返回上一個死亡的地點                |               |
| /gmp mailbox               | 打開你的郵箱，你在全球市場下架的東西都會存放在這裡 |               |
| /gmp market gui            | 打開全球市場                    |               |
| /gmp market sell <價格> <數量> | 將你手上的物品以一定的價格上架到全球市場      |               |
| /msg                       | 給某位玩家發送私聊                 |               |
| /ping                      | 查看你的延遲                    |               |
| /playtime                  | 查看你的在線時長                  |               |
| /playtimetop               | 查看伺服器在線時長排行榜              |               |
| /removehome <你的家名稱>        | 刪除家                       |               |
| /res help                  | 獲取領地插件的幫助                 |               |
| /rtp                       | 隨機傳送                      | 範圍12000*12000 |
| /sell                      | 賣出手上的物品獲得餘額               |               |
| /spawn                     | 傳送到主城                     |               |
| /tpa                       | 申請傳送到其他玩家處                |               |
| /tpahere                   | 申請其他玩家傳送到你這裡               |               |
| /worth                     | 查詢手上物品的出售價值               |               |
| /worthlist                 | 查詢所有可出售物品的價值              |               |
