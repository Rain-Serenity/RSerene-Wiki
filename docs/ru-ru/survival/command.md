# Список команд
| Команда                                          | Описание                                                                 | Примечание         |
|--------------------------------------------------|--------------------------------------------------------------------------|--------------------|
| /back                                            | Вернуться на последнее сохранённое место                                 |                    |
| /balance                                         | Проверить свой баланс                                                    |                    |
| /baltop                                          | Посмотреть топ баланса                                                   |                    |
| /dback                                           | Вернуться на место последней смерти                                      |                    |
| /gmp mailbox                                     | Открыть почтовый ящик; предметы, снятые с глобального рынка, хранятся здесь |                    |
| /gmp market gui                                  | Открыть глобальный рынок                                                 |                    |
| /gmp market sell &lt;цена&gt; &lt;количество&gt; | Выставить предмет из руки на глобальный рынок по указанной цене           |                    |
| /msg                                             | Отправить личное сообщение игроку                                        |                    |
| /ping                                            | Проверить задержку                                                       |                    |
| /playtime                                        | Проверить общее время игры                                               |                    |
| /playtimetop                                     | Посмотреть топ игроков по времени игры                                   |                    |
| /removehome &lt;название дома&gt;                | Удалить дом                                                              |                    |
| /res help                                        | Получить помощь по плагину территорий                                    |                    |
| /rtp                                             | Случайная телепортация                                                   | Радиус: 12000*12000 |
| /sell                                            | Продать предмет из руки за баланс                                        |                    |
| /spawn                                           | Телепортироваться в главный город                                        |                    |
| /tpa                                             | Запросить телепортацию к другому игроку                                  |                    |
| /tpahere                                         | Запросить телепортацию другого игрока к вам                              |                    |
| /worth                                           | Проверить стоимость продажи предмета в руке                              |                    |
| /worthlist                                       | Посмотреть стоимость всех продаваемых предметов                          |                    |
