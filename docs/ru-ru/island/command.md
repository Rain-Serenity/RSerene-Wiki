# Список команд
| Команда           | Описание              | Примечание |
|-------------------|-----------------------|------------|
| /rim tips         | Показать подсказки    |
| /rim recipemenu   | Показать особые рецепты |
| /rim slime        | Найти слизистые чанки |
| /plot auto        | Получить свой остров  |
| /plot home        | Вернуться на свой остров |
| /plot setbiome    | Установить биом острова |
| /plot sethome     | Установить точку телепорта на острове |
