# Основные настройки
## Базовая конфигурация сервера
**Этот сервер использует ядро [Lophine](https://GitHub.com/LophineLabs/Lophine).** <br>
**Вы можете узнать о возможностях сервера, просмотрев [файл конфигурации сервера](https://GitHub.com/Rain-Serenity/ServerConfig).** <br>

**Каждый остров имеет размер 512×512**
**Биом острова можно изменить (подробнее см. список команд)**

## Система сна
**Реализовано через плагин [NoPhantomSleep](https://GitHub.com/Rain-Serenity/NoPhantomSleep). При нажатии правой кнопкой мыши на кровать ночью это засчитывается как проведённая ночь, и фантомы не появляются вокруг вас.** <br>

## Получение подсолнухов
**В биоме подсолнуховых равнин использование костной муки на травяных блоках с вероятностью 10% заменяет 30% цветов и травы в области 9×9 на подсолнухи.**
## Получение кораллов
**В биоме тёплого океана использование костной муки на подводных блоках может случайным образом создавать кораллы (блоки, веера)** <br>
**Вероятности:**
| Предмет           | Вероятность |
|-------------------|-------------|
| Морская трава     | 60%         |
| Коралловый блок   | 15%         |
| Коралловое растение | 15%       |
| Коралловый веер   | 10%         |
## Естественное появление стражей
**В биоме холодного глубокого океана появление кальмаров/трески/лосося/иглобрюха/тропических рыб автоматически заменяется на стражей.**
## Поддержка JEI
**Этот сервер использует кастомный плагин [JEIBukkit](https://GitHub.com/Rain-Serenity/JEIBukkit), который автоматически синхронизирует рецепты при вашем подключении.**

## Посадка где угодно
**Крадитесь у края блока, пока не услышите звук, затем отпустите, чтобы сесть.**
## Лифт из железных блоков и рельсовый телепорт
**Лифт из железных блоков: разместите железные блоки на разных уровнях Y с одинаковыми координатами XZ, оставив достаточно места сверху. Крадитесь на блоке, чтобы телепортироваться вниз, прыгните, чтобы телепортироваться вверх.**
**Рельсовый телепорт: крадитесь на рельсах, чтобы телепортироваться на другой конец рельса.**
## Совместное развитие
**Сервер поддерживает совместное развитие между друзьями.** <br>
**/plot trust для приглашения друзей на ваш остров** <br>
**/plot visit для посещения островов других игроков**
