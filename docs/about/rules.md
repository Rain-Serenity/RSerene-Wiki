# 服务器管理条例

## I. 总条例

1. **公益属性**
    * 雨润服务器 是一个 Minecraft Java 版公益服务器。

2. **赞助原则**
    * 本服务器不包含任何付费项目，但接受玩家自愿且无条件的捐助。

3. **现实交易免责**
    * 本服务器不限制玩家在现实世界中对游戏内物品进行有偿交易（RMT），但对玩家之间引起的任何现实经济纠纷不承担任何法律与连带责任。

4. **矛盾处理**
    * 如果在游玩过程中遇到任何问题、矛盾或争吵，请冷静、理性地对待，并在必要时联系管理员介入处理。

5. **玩家公约**
    * 请尊重并友善地对待每一个人。

6. **年龄建议**
    * 根据 Minecraft 官方建议，最低游玩年龄为 9 岁以上。如果您低于此年龄，请在监护人陪同下游玩。

## II. 严厉禁止的行为

::: warning 违规处罚提示
以下行为一经发现，管理团队将根据情节严重程度给予 **警告、暂封（Temp-Ban）或踢出** 处罚。
:::

#### 1. 社区言论规范

* **发表不当言论**
    * 发表歧视、攻击性或严重不当的言论。*(详情请见下文第 III 节)*
* **不良内容**
    * 发布色情、令人反感或过于暴力的内容。
* **恶意刷屏**
    * 发送垃圾信息或恶意刷屏。
* **违规行为**
    * 冒充他人、发布未经许可的广告或进行其他不当行为。

#### 2. 游戏内行为规范

**⚖️ 作弊行为**
* a. 使用作弊客户端、外挂或违规模组（Mods）。
* b. 恶意利用服务器或游戏本身的漏洞（Bug）。

**⚔️ 恶意破坏**
* a. 未经许可破坏或篡改他人的建筑与劳动成果。
* b. 未经许可拿取、偷盗他人容器内的物品。
* c. 恶意 PvP（在非允许区域或未经双方同意强行击杀玩家）。

**🛠️ 滥用机制**
* a. 滥用插件功能（如利用领地插件圈占他人建筑、在他人箱子上强行创建锁保护）。
* b. 故意不断重复发送相同的指令（指令刷屏）。

**🏗️ 违规建筑**
* 建造任何不雅、违规或令人反感的建筑。
