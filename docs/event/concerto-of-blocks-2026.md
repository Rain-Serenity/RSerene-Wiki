## 🎯 活动主题与定位

**「方块协奏曲」原版音乐节** —— 要求玩家**仅使用游戏原版机制**在游戏内呈现音乐作品，评比"用音符盒 + 红石 + 游戏自带音效"能做出多惊艳的演出。

为什么限定原版？雨润本身就是**纯公益、纯原版生存**服务器，不篡改玩法，这次音乐节正是要把"原版能做到什么程度"推到极致。

**原版音乐的三大武器**：

1. **音符盒（Note Block）**：木板+红石粉合成，右键升半音，共 **25 档音高（F#3 → F#5 两个八度）**，上方必须是空气才能发声，声音最远传播 48 格  
   **音色由下方方块决定**：木质→贝斯、沙子/沙砾→小鼓、玻璃→点击、石头→大鼓、金块→铃铛、粘土块→长笛、浮冰→管钟、羊毛→吉他、骨块→木琴、铁块→颤音琴、灵魂沙→牛铃、南瓜→迪吉里杜管、绿宝石块→方波、干草块→班卓琴、萤石→电钢琴、泥土/空气→钢琴/竖琴
2. **红石演奏**：用按钮、压力板、拉杆、红石火把、中继器（延迟=节奏）控制电路，让多个音符盒按顺序发声形成旋律与和弦
3. **原版音效**：26.2 新增的**硫磺立方体**吸收了不同方块会改变物理特性，虽然不能直接发声，但可以作为舞台机械 / 互动装置的一部分，当然你也可以选择其他原版方块音效。
