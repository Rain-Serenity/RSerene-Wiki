# コマンド一覧
| コマンド                                         | 機能                                                                     | 備考               |
|--------------------------------------------------|--------------------------------------------------------------------------|--------------------|
| /back                                            | 最後に保存した場所に戻る                                                  |                    |
| /balance                                         | 残高を確認する                                                            |                    |
| /baltop                                          | 残高ランキングを表示する                                                  |                    |
| /co i                                           | ブロック記録照会モードをオン(オフ)にする                    |                    |
| /dback                                           | 最後に死亡した場所に戻る                                                  |                    |
| /gmp mailbox                                     | メールボックスを開く；グローバルマーケットから取り下げたアイテムが保管される |                    |
| /gmp market gui                                  | グローバルマーケットを開く                                                |                    |
| /gmp market sell &lt;価格&gt; &lt;数量&gt;       | 手に持っているアイテムを指定価格でグローバルマーケットに出品する            |                    |
| /money                                           | 残高を確認する                                                            |                    |
| /msg                                             | プレイヤーにプライベートメッセージを送信する                               |                    |
| /pay &lt;プレイヤー名&gt; &lt;金額&gt;         | プレイヤーに送金する                          | 手数料3%            |
| /ping                                            | レイテンシを確認する                                                      |                    |
| /playtime                                        | 合計プレイ時間を確認する                                                  |                    |
| /playtimetop                                     | サーバーのプレイ時間ランキングを表示する                                   |                    |
| /removehome &lt;家の名前&gt;                     | 家を削除する                                                              |                    |
| /res help                                        | 保護プラグインのヘルプを表示する                                          |                    |
| /rtp                                             | ランダムテレポート                                                        | 範囲: 24000*24000 |
| /tpsbar                                          | 自分のいるエリアのTPS、MSPTなどの表示バーを切り替える                       |                    |
| /sell gui                                        | 売却メニューを開く                                                    | 売却可能なアイテムは「基本設定」を参照 |
| /spawn                                           | メインシティにテレポートする                                              |                    |
| /tpa                                             | 他のプレイヤーにテレポートをリクエストする                                 |                    |
| /tpahere                                         | 他のプレイヤーに自分の場所へのテレポートをリクエストする                    |                    |
| /worth                                           | 手に持っているアイテムの売却価格を確認する                                 |                    |
| /worthlist                                       | すべての売却可能アイテムの価格を表示する                                   |                    |
