## 🎯 イベントテーマとコンセプト

**「ブロック協奏曲」バニラ音楽祭** — プレイヤーは**ゲームのバニラ機構のみを使用**して音楽作品を発表します。音符ブロック + レッドストーン + ゲーム内サウンドでどこまで驚くべきパフォーマンスができるかを競います。

なぜバニラ限定なのか？RSereneは**純粋な公益・純バニラサバイバル**サーバーであり、ゲームプレイを改変しません。この音楽祭は「バニラでどこまでできるか」を極限まで追求するものです。

**バニラ音楽の三大要素**：

1. **音符ブロック（Note Block）**：木材+レッドストーンダストで作成。右クリックで半音上げます。**25音程（F#3 → F#5、2オクターブ）**。上に空気が必要で、音は最大48ブロックまで届きます。  
   **音色は下のブロックで決まります**：木材→ベース、砂/砂利→スネアドラム、ガラス→クリック、石→バスドラム、金ブロック→ベル、粘土→フルート、氷塊→チャイム、羊毛→ギター、骨ブロック→シロフォン、鉄ブロック→ビブラフォン、ソウルサンド→カウベル、カボチャ→ディジュリドゥ、エメラルドブロック→矩形波、干草の俵→バンジョー、グロウストーン→エレピ、土/空気→ピアノ/ハープ
2. **レッドストーン演奏**：ボタン、感圧板、レバー、レッドストーントーチ、リピーター（遅延=リズム）で回路を制御し、複数の音符ブロックを順番に鳴らしてメロディーとコードを形成します
3. **バニラサウンド**：26.2で追加された**硫黄キューブ**はブロックを吸収して物理特性を変えます。直接音を出すことはできませんが、ステージ機構やインタラクティブ装置として活用できます。
