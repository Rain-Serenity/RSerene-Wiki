# 基本設定
## サーバーコア
::: danger 警告
機械を密集させすぎないでください。ご協力ありがとうございます！
:::
**このサーバーは [Paper](https://GitHub.com/PaperMC/Paper) コアを使用しています。** <br>
**複製に関連するすべての機能は無効化されています。** <br>

## 死亡時のアイテムロストなし
**死亡時のアイテムドロップを無効にしています。**

## 睡眠システム
**睡眠加速メカニズムとカスタム睡眠システムの両方を採用しており、[NoPhantomSleep](https://GitHub.com/Rain-Serenity/NoPhantomSleep) プラグインで実装されています。夜間にベッドを右クリックすると、一晩中寝たものとみなされ、周囲にファントムが出現しなくなります。** <br>
**30%のプレイヤーがベッドに入ると夜間加速が開始されます。**

## 視距離
**視距離は12チャンクに設定されています。**

## 土地保護
**土地保護の作成ツールは木のクワに設定されています。最大保護範囲は512×512 (X×Z)、高さ制限はありません（-64〜328まで可能）。**
