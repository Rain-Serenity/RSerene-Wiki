# 指令列表
| 指令名称            | 功能                          | 备注 |
|-----------------|-----------------------------|----|
| /rim tips       | 查看小提示                       |    |
| /rim recipemenu | 查看特殊合成表                     |    |
| /rim slime      | 查看史莱姆区块                     |    |
| /tpsbar         | 开启(关闭)查看你所在区域的TPS、MSPT等的悬浮条 |    |
| /plot auto      | 领取你的岛屿                      |    |
| /plot home      | 回到你的岛屿                      |    |
| /plot setbiome  | 设置你的岛屿的生物群系                 |    |
| /plot sethome   | 设置你的岛屿传送位置                  |    |
