# 基本设定
## 服务器基本配置
::: danger 警告
机器请勿过于密集，谢谢配合！
:::
**本服务器开服配置为 Intel Xeon Platinum 8270 Processor 8核心 + 40G内存** <br>
**本服务器采用的是[Paper](https://GitHub.com/PaperMC/Paper)核心。** <br>
**启用了所有原版复制特性** <br>
## 岛屿设定
**每个人的岛屿都为512×512**
**岛屿生物群系可以切换(指令详见指令列表)**

## 睡眠系统
**使用[NoPhantomSleep](https://GitHub.com/Rain-Serenity/NoPhantomSleep)插件实现。具体设定如下，只要你在夜晚右键床，就会判定你已经度过了整个夜晚，不会在你周围刷新幻翼。** <br>
