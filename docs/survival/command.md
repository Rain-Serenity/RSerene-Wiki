# 指令列表
| 指令名称                       | 功能                        | 备注             |
|----------------------------|---------------------------|----------------|
| /back                      | 返回上一个保存的地点                |
| /balance                   | 查看你的余额                    |                |
| /baltop                    | 查看余额排行榜                   |                |
| /co i                      | 启动(关闭)方块记录查询模式            |
| /dback                     | 返回上一个死亡的地点                |                |
| /gmp mailbox               | 打开你的邮箱，你在全球市场下架的东西都会存放在这里 |                |
| /gmp market gui            | 打开全球市场                    |                |
| /gmp market sell <价格> <数量> | 将你手上的物品以一定的价格上架到全球市场      |                |
| /money                     | 查看你的余额                    |
| /msg                       | 给某位玩家发送私聊                 |                |
| /pay <玩家名> <金额>            | 给某位玩家转账                   | 手续费3%          |
| /ping                      | 查看你的延迟                    |                |
| /playtime                  | 查看你的在线时长                  |                |
| /playtimetop               | 查看服务器在线时长排行榜              |                |
| /removehome <你的家名称>        | 删除家                       |                |
| /res help                  | 获取领地插件的帮助                 |                |
| /rtp                       | 随机传送                      | 范围24000*24000  |
| /tpsbar                    | 查看你所在区域的TPS、MSPT等         |                |
| /sell gui                  | 打开出售菜单                    | 能出售的物品详见"基本设定" |
| /spawn                     | 传送到主城                     |                |
| /tpa                       | 申请传送到其他玩家处                |                |
| /tpahere                   | 申请其他玩家传送到你这               |                |
| /worth                     | 查询手上物品的出售价值               |                |
| /worthlist                 | 查询所有可出售物品的价值              |                |
