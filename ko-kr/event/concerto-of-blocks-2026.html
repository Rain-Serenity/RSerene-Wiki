블록 협주곡

지원 버전: Minecraft Java Edition 26.2 (Chaos Cubed)
서버: RSerene ｜ IP: Play.RSerene.com ｜ 바닐라 서바이벌 또는 스카이블록
심사일: 2026년 8월 16일
본 이벤트는 Minecraft 공식 이벤트가 아니며 Mojang / Microsoft와 관련이 없습니다.

🎯 이벤트 테마 및 콘셉트

「블록 협주곡」바닐라 음악 축제 — 참가자는 게임의 바닐라 메커니즘만 사용하여 게임 내에서 음악 작품을 선보입니다. Note Block + Redstone + 게임 내장 사운드로 얼마나 놀라운 퍼포먼스를 만들 수 있는지 겨루는 대회입니다.

왜 바닐라만 고집하나요? RSerene은 순수 공익, 순수 바닐라 서바이벌 서버로, 게임플레이를 변조하지 않습니다. 이 음악 축제는 "바닐라로 얼마나 할 수 있는가"를 극한까지 끌어올리는 데 목적이 있습니다.

바닐라 음악의 세 가지 요소:

  1. Note Block: 나무 판자+레드스톤 가루로 제작. 우클릭으로 반음 올림. 25음높이(F#3 → F#5, 2옥타브). 위에 공기가 있어야 소리가 나며, 최대 48블록까지 전달됩니다.
    음색은 아래 블록에 따라 결정: 나무→베이스, 모래/자갈→스네어 드럼, 유리→클릭, 돌→베이스 드럼, 금 블록→벨, 점토→플루트, 꽁꽁 언 얼음→차임, 양털→기타, 뼈 블록→실로폰, 철 블록→비브라폰, 영혼 모래→카우벨, 호박→디저리두, 에메랄드 블록→구형파, 건초 더미→밴조, 발광석→일렉트릭 피아노, 흙/공기→피아노/하프
  2. 레드스톤 연주: 버튼, 압력판, 레버, 레드스톤 횃불, 리피터(지연=리듬)로 회로를 제어하여 여러 Note Block이 순서대로 소리를 내 멜로디와 코드를 만듭니다
  3. 바닐라 사운드 효과: 26.2에 추가된 황산염 큐브는 블록을 흡수하여 물리적 특성이 변합니다. 직접 소리를 낼 수는 없지만, 무대 기계/인터랙티브 장치의 일부로 활용할 수 있습니다.

팁

바닐라 서바이벌은 32000블록 이후에 건축하는 것을 권장합니다（성능 향상을 위해）
스카이블록은 새로운 섬을 생성하여 건축하는 것을 권장합니다（성능 향상을 위해）

작품 필수 요구사항

  1. 100% 바닐라: Note Block, 레드스톤 부품, 게임 내장 사운드(주크박스로 바닐라 레코드 재생, 몹 소리 등)만 사용 가능. Mod, 플러그인, 외부 오디오 파일, 사운드를 대체하는 리소스팩은 금지
  2. 공연 시간: 30초 ~ 5분
  3. 작품은 26.2 바닐라 환경에서 완전히 재생 가능해야 함
  4. 오리지널 곡을 권장합니다. 기존 곡을 커버하는 경우, 참가자는 관련 음악 사용에 대한 허가를 직접 받아야 합니다
  5. 무대 제작에는 바닐라 블록을 사용한 시각적 장식이 허용됩니다

금지 행위

  • 사운드 변조 MOD나 리소스팩 사용
  • 타인의 레드스톤 회로를 무단 도용
  • 다른 참가자에 대한 악의적인 방해
  • 위 사항 위반 시 즉시 실격 처리되며 서버 규정에 따라 조치됩니다

  2. 음악 저작권: 기존 곡을 커버하는 경우, 참가자는 원작자/저작권자의 허가를 직접 받아야 합니다. 오디오 저작권으로 인한 분쟁, 게시 중단 또는 기타 결과에 대해 참가자 본인이 전적인 책임을 지며, RSerene 서버 및 운영자는 어떠한 책임도 지지 않습니다.
